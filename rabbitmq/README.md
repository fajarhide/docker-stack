# RabbitMQ in docker-compose

## Play
Open [http://192.168.59.103:15672/](http://192.168.59.103:15672/) (or what ever IP you get when you run `docker-machine ip`)

```
open http://$(docker-machine ip default):15672/
```
and use the login

```
username: rabbitmq
password: rabbitmq
```
